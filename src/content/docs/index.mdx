Welcome to the Gnosis API documentation. Gnosis provides a REST API for accessing a comprehensive biblical knowledge graph including people, places, events, cross-references, Strong's concordance, and more.

## Quick Links

- [Getting Started](/getting-started/) — get your API key and make your first request
- [Authentication](/authentication/) — API keys, rate limits, and tiers
- [Data Model](/data-model/) — entity types and relationships

## Endpoints

- [People](/endpoints/people/) — biblical figures with family trees
- [Places](/endpoints/places/) — locations with coordinates
- [Events](/endpoints/events/) — events with timeline data
- [Verses](/endpoints/verses/) — entity lookup and cross-references
- [Strong's](/endpoints/strongs/) — Hebrew and Greek concordance
- [Dictionary](/endpoints/dictionary/) — historical dictionary entries
- [Topics](/endpoints/topics/) — topical scripture arrangements
- [Hebrew](/endpoints/hebrew/) — word-level morphological data
- [Lexicon](/endpoints/lexicon/) — Hebrew lexicon
- [Search](/endpoints/search/) — full-text and semantic search
