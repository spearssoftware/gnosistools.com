Welcome to the Gnosis API documentation. Gnosis provides a REST API for accessing a comprehensive biblical knowledge graph including people, places, events, cross-references, Strong's concordance, and more.

## Quick Links

- [Getting Started](/docs/getting-started/) — get your API key and make your first request
- [Authentication](/docs/authentication/) — API keys, rate limits, and tiers
- [Data Model](/docs/data-model/) — entity types and relationships

## Endpoints

- [People](/docs/endpoints/people/) — biblical figures with family trees
- [Places](/docs/endpoints/places/) — locations with coordinates
- [Events](/docs/endpoints/events/) — events with timeline data
- [Verses](/docs/endpoints/verses/) — entity lookup and cross-references
- [Strong's](/docs/endpoints/strongs/) — Hebrew and Greek concordance
- [Dictionary](/docs/endpoints/dictionary/) — historical dictionary entries
- [Topics](/docs/endpoints/topics/) — topical scripture arrangements
- [Hebrew](/docs/endpoints/hebrew/) — word-level morphological data
- [Lexicon](/docs/endpoints/lexicon/) — Hebrew lexicon
- [Search](/docs/endpoints/search/) — full-text and semantic search
